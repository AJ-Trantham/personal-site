NASA SUITS Augmented Reality System

In this project I worked with an interdisciplinary team of Boise State students to design a heads-up display system within an AR environment to aid astronauts in space walks . This project was developed on the Microsoft Hololens using C# and the Unity game engine.

LEXI: A Graphical Text Editor

Lexi started as a project from an Object Oriented Design Patterns course. The goal of this project was to utilize OO Design Patterns to build the foundations of a text editor with an emphasis of reuseable code. This project was based on the Gang of Four's classic Design Patterns text.
